"pressure_list" contains the 30 pressure values that the elastic tensors are measured at.

"N16_pbc" contains elastic tensors for 1000 granular packings (each generated with a different random seed) with 16 particles. The elastic tensors are measured at 30 different pressures. The reshaped array has size (1000x30x6x6). 

Similarly, "N32_pbc" contains elastic tensors for 1000 granular packings with 32 particles.
