"xtal_database_cmats" contains elastic tensors for 1181 structures from the database in de Jong et. al. (2015) [https://www.nature.com/articles/sdata20159]

The reshaped data is 1181x6x6 (each 6x6 matrix is the full elastic tensor)

"xtal_network_pressure" contains the 20 pressure values that the spring network elastic tensors are measured at.

"xtal_network_cmats" contains elastic tensors for spring networks derived from the cubic crystals from the above database. The elastic tensors are measured at 20 different pressure. The reshaped array is size 448x20x6x6
